# miyagi
## Open-source format and parsing library for Japanese instrumentation and scoring

This project is both a learning project of how to use GitHub, common programming languages. This repository will be a work in progress for some time. Any help is encouraged!

### Project scope
#### 1. File format for traditional Japanese musical scores
#### 2. Parsing and rendering of scores
#### 3. Implementation of conversion to common formats such as MusicXML and MIDI
